# FreePhish
Social Engineer Tool / Fake Pages Maker / .... etc 
#Installation
- Termux :
$ git clone https://Github.com/MiD370/FreePhish
$ cd FreePhish
$ chmod +x setup-termux.sh
$ ./setup-termux.sh
$ python main.py
- Linux-Desktop :
$ git clone https://Github.com/MiD370/FreePhish
$ cd FreePhish
$ chmod +x setup.sh
$ sudo setup.sh
$ sudo python3 main.py
